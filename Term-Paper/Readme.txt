Abstract

Today, many software programs are created as web-based apps that can be accessed using an Internet browser. Selenium is a collection of strong software tools that interact with a variety of browsers, operating systems, programming languages, and testing frameworks. The framework architecture and framework implementation will be discussed in this term paper.
